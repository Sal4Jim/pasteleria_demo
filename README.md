# Pastelería Demo

Un sistema de punto de venta y gestión visual para una pastelería moderna. Este proyecto cuenta con un entorno interactivo y dashboards adaptables.

## Tecnologías Principales

- **Framework**: React + Vite
- **Lenguaje**: TypeScript
- **Estilos**: Tailwind CSS + Shadcn UI
- **Base de Datos / Backend**: Supabase

## Setup Local

Asegúrate de tener instalado Node.js, luego corre los siguientes comandos desde la terminal:

```sh
# Instala las dependencias del proyecto
npm install

# Inicia el entorno de desarrollo
npm run dev
```

El proyecto estará disponible localmente en `http://localhost:8080`.

## Scripts Útiles

- `npm run dev`: Inicia el servidor de desarrollo local.
- `npm run build`: Compila el código para producción en el directorio `dist`.
- `npm run lint`: Ejecuta el validador de código estático (ESLint).
- `npm run preview`: Previsualiza la compilación antes de mandarla a producción.
